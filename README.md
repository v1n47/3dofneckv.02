3DoF Robotic Head (My First Project! 🚀)
Welcome to the repository for my very first hardware and robotics project! This repository contains the design, firmware, and code for a 3-Degrees-of-Freedom (3DoF) Robotic Head.

Built as a hands-on introduction to mechanical assembly, motor control, and embedded systems, this project can mimic 
basic neck movements: Pitch (tilt up/down), Yaw (turn left/right), and Roll (tilt side-to-side).
